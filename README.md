Wein2D library family

Wein2D is a collection of libraries for simple access to 2D graphics, user input and audio.
It has multiple implementations of similar classes and method calls for different platforms,
even spanning across languages. This makes it easy to port between platforms, but still
requires some time.

Check out all libraries here:
https://wein2ddocs.netlify.app/

Check out games made with Wein2D:
[Wein2D][Wein2DAndroid] https://devtaube.itch.io/serverless
[Wein2D.js-WASM]        https://devtaube.itch.io/isla-fuego
[Wein2D.js-WASM]        https://devtaube.itch.io/colorlift
